Über KMR

KMR ist eine dezentrale Kryptowährung, die darauf abzielt, digitale Transaktionen zu revolutionieren. Mit einem Fokus auf Sicherheit, Skalierbarkeit und Community-Beteiligung haben wir eine Plattform geschaffen, die es jedem ermöglicht, Teil der Blockchain-Revolution zu werden.

Unsere Errungenschaften

  • Erfolgreicher Start des KMR-Netzwerks
  • Integration von Mining-Funktionen für die Community
  • Entwicklung einer benutzerfreundlichen Wallet
  • Aufbau einer wachsenden Community auf GitHub und Twitter

So steigst du ein

Folge diesen einfachen Schritten, um Teil des KMR-Netzwerks zu werden:

  1. Klone das Repository von GitHub: https://github.com/Ser0101/Komara-network.git
  2. Navigiere in das Verzeichnis: cd Komara-network
  3. Baue das Projekt: cargo build --release
  4. Starte den Mining-Prozess: cargo run --release -- mine
  5. Lies das README für weitere Informationen: README.md
